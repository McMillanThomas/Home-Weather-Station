About

What is it?

This is a DIY project to measure and display temperature and windspeed. A homemade sensor unit records and uploads the data, which is accessable via web browser.

How does it work?

The weather station – pictured below – is mounted behind windmill blades.

The sensors are housed in a wooden box about 10 x 10 cm. Pulling the tab opens the box.
The box attaches with velcro behind the windmill blades.

Inside the Weather Station
- A microcontroler (Adafruit Feather Huzzah)
- A temperature sensor (Adafruit MCP9808)
- An ultrasonic sensor (HC-SR0)
- A power jack (and a potentiometer to reduce the voltage)

The electronics are held in place because the ultrasonic sensor fits tightly into the two cut holes. (A smaller, better fitting container could also be used.)
The Arduino board takes 3-6V to run.

Power
Electricity is supplied through a 5V DC Adapter. The unit saves power by sleeping and only waking every 30 seconds to record and send weather data.

Calculating the Windspeed
The windspeed is calculated using the ultrasonic sensor.


The ultrasonic sensor sends a signal every 2 milliseconds. If a windmill blade is in front of the sensor, the sensor receives and records a response. By adding how many responses the sensor recieves per 300 signals sent, the windspeed can be calculated.

windspeed = (number of blade passes) x (conversion constant)

To enable calculating even slow windspeeds – where a blade might be in front of the sensor for more than 2 milliseconds – a new response is recorded only if the previous signal did not return a response.


Data Transmition
The weather station connects to a local wifi network and transmits the windspeed, temperature and a timestamp to a MySQL database.

Website
On the website, the windmill image's blades spin faster or slower based on the actual windspeed. Every 34 seconds the website updates with new data from the database.

Code
The weather station code runs on an Arduino microcontroler, and the website was built using Bootstrap.
