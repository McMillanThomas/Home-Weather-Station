About

What is it?

This is a DIY project to measure and display temperature and windspeed. A homemade unit with sensors records and uploads the data, which is accessable to anyone via a web browser.

How does it work?

The unit – pictured below – is mounted behind windmill blades.



Inside the unit is:
- A microcontroler (Adafruit Feather Huzzah)
- A temperature sensor (Adafruit MCP9808)
- An ultrasonic sensor (HC-SR0)
- A power jack (and a potenometer to reduce the voltage)


Power comes from a 5C DC Adapter.

Every 30 seconds, the unit records data and sends it over wifi. This website displays the data here.

The windspeed is calculated using the ultrasonic sensor, which measures how many times a blade passes in front of it.

On the website, the windmill blades spin faster or slower based on the actual windspeed.

The unit's program runs on an Arduino microcontroler, and the website was built using Bootstrap.
